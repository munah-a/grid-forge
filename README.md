# GridForge GIS

Professional browser-based GIS application for gridding, contouring, and spatial visualization.

## Features

- **Point Data Import** - CSV, TSV, GeoJSON support
- **Gridding Algorithms** - IDW, Natural Neighbor, Minimum Curvature
- **Contour Generation** - Automated isoline generation with labeling
- **Interactive Visualization** - Pan, zoom, multi-layer management
- **Multiple Base Maps** - Dark, Light, Aerial, Topographic, Blueprint themes
- **Color Ramps** - Viridis, Plasma, Terrain, Ocean, and more
- **100% Client-Side** - No server required, data never leaves your browser

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
